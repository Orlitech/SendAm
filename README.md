# SendAm
Fast, private file sharing over your local network. Drag, drop, scan the QR — files move device to device without touching the cloud. Built for Naija, with English and Pidgin. No app install, no signup, no data leaves your Wi-Fi.

Africa-ready file transfer that never leaves your Wi-Fi.
SendAm turns any laptop into a local sharing hub. Run it, and every phone, tablet, and PC on the same network can send files to each other by scanning a QR code — no app to install, no account to create, no data burning through anybody's bundle.
Built for places where bandwidth is expensive and connectivity is unreliable: everything happens over the local network, so transfers run at Wi-Fi speed whether or not there's internet.
What it does

Drag-and-drop transfers between any devices on the same network
QR pairing — scan and you're connected, nothing to download
WebRTC peer-to-peer with automatic HTTP fallback
Live chat and clipboard sharing across devices
File requests — ask another device to send you something
Wi-Fi QR generator for onboarding new devices onto the network
English and Nigerian Pidgin interface
Auto-wiping storage — files are cleared on shutdown and expire on a timer
Installable PWA, dark and light themes
